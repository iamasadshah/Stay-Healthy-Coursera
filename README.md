## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18 or higher)

### Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
```

2. Navigate to the project directory:

3. Install dependencies:
